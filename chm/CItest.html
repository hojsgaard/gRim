Test for conditional independence

Description

Tests for conditional independence of two variables u and v given a third set W. These tests can be made in a contingency table or in the multivariate normal distribution.

Usage

CItest(x, set, statistic = "dev", method = "chisq", adjust = TRUE,
   slice.info = TRUE, B = 2000, n)

Arguments

x Either a contingency table (an array or a table) or an empirical covariance matrix. In the latter case the argument n must also be given
set A vector or a formula specifying (u,v,W) – the test being made is for u independent of v given W.
